SuriK-Glow is a modern, responsive web application dedicated to showcasing and retailing authentic Korean skincare and beauty products. It features a curated selection of famous K-Beauty brands, interactive product details, and a dynamic shopping cart experience.

---

## ✨ Features
- **Dynamic Product Catalog:** Filter products by categories like Skincare, Makeup, Haircare, and Perfume.
- **Detailed Product Modals:** View key ingredients, benefits, and recommendations for each product.
- **Interactive Shopping Bag:** Real-time subtotal calculation and dynamic item counters.
- **Fully Responsive Design:** Tailored experience across mobile, tablet, and desktop screens using Tailwind CSS.
- **Python Flask Backend:** Secure handling of products, dynamic APIs, and session management.

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+), FontAwesome Icons
- **Backend:** Python, Flask
- **Database:** SQLite / Flask-SQLAlchemy

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/YOUR_USERNAME/SuriK-Glow.git](https://github.com/YOUR_USERNAME/SuriK-Glow.git)
   cd SuriK-Glow

```
 2. **Run the Application:**
   ```bash
   python app.py
   
   ```
   Open http://127.0.0.1:5000 in your web browser.
